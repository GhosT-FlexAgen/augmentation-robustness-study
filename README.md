# Контролируемая аугментация изображений и устойчивость моделей компьютерного зрения

Данный репозиторий содержит материалы эксперимента для исследования:

**«Контролируемые трансформации изображений для повышения устойчивости и безопасности моделей компьютерного зрения»**

В работе исследуется влияние различных стратегий аугментации изображений на:

* точность классификации;
* устойчивость моделей к искажениям;
* робастность моделей искусственного интеллекта.

Эксперименты проводились на наборе данных CIFAR-10 с использованием архитектуры ResNet-18.

# Исследуемые группы аугментаций

В рамках эксперимента сравнивались следующие группы методов:

* Базовая модель (без аугментации)
* Геометрические преобразования
* Фотометрические преобразования
* Шумовые аугментации
* Структурные методы регуляризации
* Генеративные аугментации
* Комбинированные конвейеры

---

# Параметры эксперимента

* Набор данных: CIFAR-10
* Архитектура модели: ResNet-18
* Фреймворк: PyTorch
* Библиотека аугментаций: Albumentations
* Оптимизатор: AdamW
* Размер батча: 128
* Learning rate: 1e-3
* Количество эпох: 30
* Среда выполнения: Google Colab
* GPU: NVIDIA T4

---

# Оценка устойчивости модели

Помимо стандартной точности на исходных данных выполнялась проверка устойчивости модели на искажённых выборках:

* гауссов шум;
* размытие;
* искусственный туман;
* изменение яркости.

Итоговая метрика робастности вычислялась как среднее значение точности на всех искажённых наборах данных.

---

# Основные результаты

* Геометрические преобразования обеспечили максимальную точность на исходных данных.
* Шумовые и генеративные аугментации значительно повысили устойчивость модели к деградациям изображения.
* Комбинированные стратегии показали наиболее сбалансированные результаты.
* Генеративные аугментации обеспечили наибольшую среднюю робастность модели.
